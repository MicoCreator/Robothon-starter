## Project Summary

- **Project name:** AURORA FRAME 2.0 - Adaptive Bimanual Shelter Recovery Under Wind
- **Robot platform:** Fixed-base bimanual emergency-response rig with six force-limited Cartesian axes
- **Task goal:** Catch a passive spring-loaded shelter, physically engage three lock sliders, and recover the frame during calm, crosswind, rotating-gust, and microburst wind profiles.
- **Technical approach:** Sensor-gated bimanual phase policy using frame hinge residual, spatial-tendon lengths, latch position, and palm-handle contact force.
- **Core evidence:** Generated MP4, self-contained MJCF, 128 paired seeded trials, four wind profiles, and machine-readable metric/trajectory artifacts.
- **Verified result:** Generated by `python run_aurora_frame.py --cases 128 --demo-seed 4118 --demo-scenario microburst_recovery` and audited by `python verify_submission.py`.
- **Current limits:** MuJoCo-only; external force/torque wind profile rather than CFD; fixed-base robot; tendon rather than deformable-rope model.

## How to run

```bash
python -m pip install -r requirements.txt
python run_aurora_frame.py --cases 128 --demo-seed 4118 --demo-scenario microburst_recovery
python verify_submission.py
```
