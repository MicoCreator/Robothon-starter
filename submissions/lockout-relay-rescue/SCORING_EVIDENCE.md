# Scoring Evidence — Lockout Relay Rescue

All of the following artifacts were generated by running:

```bash
python run_lockout_rescue.py --cases 36
```

## Judge-readable proof

| Rubric area | Evidence |
|---|---|
| Reproducibility | One Python command generates the scene, MP4, benchmark, trajectory, and reports. `verify_submission.py` checks the finished package. |
| MuJoCo depth | Generated MJCF includes a movable panel, 15 finger joints, Cartesian wrist joints, contacts, friction, gravity, a spring-return latch, a rotary hinge, tactile sites, pose sensors, and actuators. |
| Task design | The task is a complete safety sequence: grasp the correct relay, isolate it, press the verification latch, seat a lockout tag, then verify terminal state. |
| Control | A phase policy uses marker-pose residual correction, tactile contact gating, a tactile overload back-drive, and wrist-intent torque for the rotary isolator. |
| Dexterity | Five independently commanded fingers use 15 physical joints and five tactile contact sites to establish the isolation gate. |
| Engineering quality | `metrics.json`, `paired_ablation.json`, `benchmark.csv`, `demo_trajectory.json`, `sensor_policy_card.json`, and `artifact_manifest.json` are machine-readable outputs. |
| Presentation | `lockout_relay_demo.mp4` is a 67.7-second generated video with live phase, contact, alignment, isolator-angle, latch, and safety overlays. |
| Innovation | The benchmark evaluates the real safety value of feedback by comparing it to an identical-seed open-loop baseline during panel motion. |

## Verified results

- Feedback controller: **36/36** safe-lockout completions (**100.0%**).
- Open-loop controller: **2/36** completions (**5.6%**) on the same 36 seeded disturbances.
- Improvement: **94.4 percentage points**.
- Mean final isolator angle: **86.9°** with feedback versus **38.8°** open loop.
- Mean tag/slot error: **14.1 mm** with feedback versus **42.9 mm** open loop.

## Integrity notes

- The visual signal is a noisy calibrated marker observation, not an end-to-end learned camera model.
- The rotary joint is commanded through a disclosed contact-gated torque proxy; the code does not set the switch pose directly.
- The lockout tag is an attached end-effector tool, not a free object grasped by the hand.
- This is a MuJoCo simulation, not a hardware-validation claim.
